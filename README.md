# Snake

Ce projet a été réalisé en groupe dans le cadre du **Travail Pratique 1 (TP1)** du cours **réalisation d'un jeu vidéo**, offert à l’**Université du Québec à Chicoutimi (UQAC)**.

---

- **Titre du projet :** Snake
- **Date de réalisation :** 31 janvier 2026 
- **Type de projet :** Travail de groupe  
- **Moteur de développement :** Unity 6000.3.5f1

---

## Objectif du projet

Ce jeu est développé sur Unity et reprend le concept classique du **Snake Game**. Il conserve les fonctionnalités de base du jeu original tout en y ajoutant des bonus et des malus influençant la vitesse du serpent. Un système de leaderboard est également intégré, permettant d’enregistrer les meilleurs scores, même lorsque l’application est fermée.

## Structure du projet

- `Code/Asset/Script/` : Contient le code C# du projet  
- `Code/Asset/image/` : Contient les images utilisé  
- `Code/Asset/Scene/` : Contient les differente scenes du jeu
- `Code/Asset/Sprite/` : Contient le corps du snake ainsi que les fruits
- `README.md` : Présentation et documentation du projet

---

## Remarque

Ce projet a été développé exclusivement dans un but académique et ne constitue pas un produit fini.  
Il s’agit d’une **preuve de concept** visant à mettre en application les principes de base de la realisation de jeu vidéo, avec un gameplay de type Snake Game.
